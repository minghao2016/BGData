Applies a function on each row or column of a matrix in parallel.

Usage

parallelApply(X, MARGIN, FUN, nTasks = parallel::detectCores(), mc.cores = parallel::detectCores(), ...)

Arguments

X
A matrix.
MARGIN
The subscripts which the function will be applied over. 1 indicates rows, 2 indicates columns.
FUN
The function to be applied.
nTasks
The number of submatrices of X to be processed in parallel.
mc.cores
The number of cores (passed to mclapply).
...
Additional arguments to be passed to apply.

Description

The input matrix X is broken into nTasks chunks and passed to mclapply. The number of cores can be configured using mc.cores. Uses apply from base internally.
