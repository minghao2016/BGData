Converts a BEDMatrix object to a BGData object.

Usage

"as.BGData"(x, alternatePhenotypeFile = NULL, ...)

Arguments

x
A BEDMatrix object.
alternatePhenotypeFile
Path to an alternate phenotype file.
...
Additional arguments to the BEDMatrix constructor (currently unused).

Value

A BGData object.

Description

If a FAM file (which corresponds to the first six columns of a PED file) of the same name and in the same directory as the BED file exists, the @pheno slot will be populated with the data stored in that file. Otherwise a stub that only contains an IID column populated with the rownames of @geno will be generated.

Details

PED only allows for a single phenotype. If more phenotypes are required it is possible to store them in an alternate phenotype file. The path to such a file can be provided with alternatePhenotypeFile and will be merged with the data in the @pheno slot.
