Computes a genomic relationship matrix G=xx' without ever loading G in RAM by creating a symDMatrix.

Usage

getG.symDMatrix(X, i = seq_len(nrow(X)), j = seq_len(ncol(X)), centerCol = TRUE, centers = NULL, scaleCol = TRUE, scales = NULL, scaleG = TRUE, folder = randomString(), vmode = "double", saveRData = TRUE, bufferSize = NULL, nBuffers = NULL, nTasks = nCores, nCores = 1, verbose = TRUE)

Arguments

X
A matrix-like object, typically @geno of a BGData object.
i
(integer, boolean or character) Indicates which rows should be used. By default, all rows are used.
j
(integer, boolean or character) Indicates which columns should be used. By default, all columns are used.
centerCol
TRUE/FALSE whether columns must be centered before computing xx'.
centers
Precomputed centers.
scaleCol
TRUE/FALSE whether columns must be scaled before computing xx'.
scales
Precomputed scales.
scaleG
TRUE/FALSE whether xx' must be scaled.
folder
Folder in which to save the symDMatrix.
vmode
vmode of ff objects.
saveRData
Whether to save an RData file to easily reload symDMatrix
bufferSize
The number of columns of x that are brought into RAM for processing. Overwrites nBuffers. If both parameters are NULL, all elements of j are used.
nBuffers
The number of partitions of the columns of x that are brought into RAM for processing. Is overwritten by bufferSize. If both parameters are NULL, all elements of j are used.
nTasks
The number of chunks that each chunk is split into for processing in parallel.
nCores
The number of cores (passed to mclapply).
verbose
If TRUE more messages are printed.

Value

A positive semi-definite symmetric numeric matrix.

Description

Offers options for centering and scaling the columns of x before computing xx'.
