Computes a genomic relationship matrix G=xx' as a symDMatrix.

Usage

getG.symDMatrix(X, nChunks = 5, chunkSize = NULL, centers = NULL, scales = NULL, centerCol = T, scaleCol = T, nChunks2 = 1, folder = randomString(), vmode = "double", verbose = TRUE, saveRData = TRUE, mc.cores = parallel::detectCores(), scaleG = T)

Arguments

X
A matrix-like object, typically @geno of a BGData object.
nChunks
The number of columns that are processed at a time.
chunkSize
The number of columns that are processed at a time.
centers
Precomputed centers.
scales
Precomputed scales.
centerCol
TRUE/FALSE whether columns must be centered before computing xx'.
scaleCol
TRUE/FALSE whether columns must be scaled before computing xx'.
nChunks2
The number of chunks that each chunk is split into for processing in parallel.
folder
Folder in which to save the symDMatrix.
vmode
vmode of ff objects.
verbose
If TRUE more messages are printed.
saveRData
Whether to save an RData file to easily reload symDMatrix
mc.cores
The number of cores (passed to mclapply).
scaleG
TRUE/FALSE whether columns must be scaled before computing xx'.

Value

A positive semi-definite symmetric numeric matrix.

Description

Offers options for centering and scaling the columns of x before computing xx'.
