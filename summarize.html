Calculates frequencies of missing values and alleles.

Usage

summarize(X, verbose = FALSE, bufferSize = 5000, nTasks = parallel::detectCores(), mc.cores = parallel::detectCores())

Arguments

X
A matrix-like object, typically @geno of a BGData object.
verbose
If TRUE more messages are printed.
bufferSize
Represents the number of columns of @geno that are brought into RAM for processing (5000 by default).
nTasks
Represents the number of parallel tasks each buffer is split into.
mc.cores
The number of cores (passed to mclapply).

Description

Calculates frequencies of missing values and alleles.
