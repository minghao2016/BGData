Loads BGData (and other) objects from .RData files.

Usage

load.BGData(file, envir = parent.frame())

Arguments

file
The name of the .RData file to be loaded.
envir
The environment where to load the data.

Description

This function is similar to load, but also initializes the different types of objects that the @geno slot of a BGData object can take. Currently supported are ff_matrix, big.matrix, and BEDMatrix objects. If the object is of type LinkedMatrix, all nodes will be initialized with their appropriate method.
