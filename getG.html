Computes a genomic relationship matrix G=xx'.

Usage

getG(x, nChunks = ceiling(ncol(x)/10000), scaleCol = TRUE, centerCol = TRUE, scaleG = TRUE, verbose = TRUE, i = seq_len(nrow(x)), j = seq_len(ncol(x)), i2 = NULL, minVar = 1e-05, nChunks2 = parallel::detectCores(), scales = NULL, centers = NULL, saveG = FALSE, saveType = "RData", saveName = "Gij", mc.cores = parallel::detectCores())

Arguments

x
A matrix-like object, typically @geno of a BGData object.
nChunks
The number of columns that are processed at a time.
scaleCol
TRUE/FALSE whether columns must be scaled before computing xx'.
centerCol
TRUE/FALSE whether columns must be centered before computing xx'.
scaleG
TRUE/FALSE whether xx' must be scaled.
verbose
If TRUE more messages are printed.
i
(integer, boolean or character) Indicates which rows should be used. By default, all rows are used.
j
(integer, boolean or character) Indicates which columns should be used. By default, all columns are used.
i2
(integer, boolean or character) Indicates which rows should be used to divide matrix into blocks.
minVar
Columns with variance lower than this value will not be used in the computation (only if scaleCol is set).
nChunks2
The number of chunks that each chunk is split into for processing in parallel.
scales
Precomputed scales if i2 is used.
centers
Precomputed centers if i2 is used.
saveG
Whether to save genomic relationship matrix into file.
saveType
File format to save genomic relationship matrix in. Either RData or ff.
saveName
Name without extension to save genomic relationship matrix with.
mc.cores
The number of cores (passed to mclapply).

Value

A positive semi-definite symmetric numeric matrix.

Description

Offers options for centering and scaling the columns of x before computing xx'. If centerCol=FALSE, scaleCol=FALSE and scaleG=FALSE, getG produces the same outcome than tcrossprod.
