Generate and store a simulated plaintext raw PED file (see --recodeA in PLINK) or PED-like file for testing purposes.

Usage

simPED(filename, n, p, genoChars = 0:2, na.string = NA, propNA = 0.02, returnGenos = FALSE)

Arguments

filename
The path where to save the generated file.
n
The number of observations to generate.
p
The number of markers to generate.
genoChars
The alphabet used to generate the genotypes.
na.string
The symbol used to denote missing values.
propNA
The probability of generating NAs.
returnGenos
Whether to return the genotypes from the function.

Description

Generate and store a simulated plaintext raw PED file (see --recodeA in PLINK) or PED-like file for testing purposes.
